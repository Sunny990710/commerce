# E-Land AI Fashion Mall — PRD (제품 요구사항 정의서)

## 1. 문서 정보

| 항목 | 내용 |
|------|------|
| 제품명 | E-Land AI Fashion Mall |
| 형태 | 웹 SPA (React 19 + Tailwind, 단일 `index.html`) |
| 언어/시장 | 한국어 UI, 패션 이커머스·AI 추천 |

---

## 2. 제품 비전·포지셔닝

- **비전:** 자연어 대화와 AI 추천으로 원하는 스타일·상품을 빠르게 찾고, 개인 옷장·위시와 연동해 코디까지 이어지는 **AI 패션 쇼핑** 경험을 제공한다.
- **차별점(의도):** 키워드 검색을 넘어 **상황·무드·스타일**을 묻는 대화형 탐색, **필터(브랜드·색상·가격·카테고리)**로 결과 정제, **AI 옷장**에서 보유/위시 기반 코디 상담.

---

## 3. 목표 (Goals)

1. 사용자가 홈에서 질문을 입력하거나 **대화** 화면에서 탐색·필터링해 상품을 발견한다.
2. 추천 상품 **상세(슬라이드 패널)**에서 가격·옵션·혜택·리뷰·AI 탭을 확인하고, 위시·가격 알림 등 액션을 수행한다.
3. **AI 옷장**에서 내 옷·위시를 선택해 AI와 코디/스타일 대화를 나눈다.
4. (선택) 로그인 후 **프로필**로 취향을 저장하고, **체크아웃** 플로우로 구매 의도를 완료한다.

---

## 4. 확정된 범위·비목표·KPI

본 섹션은 제품·개발 기준으로 **확정**한 범위와 측정 지표이다. (이해관계자 검토용 스냅샷)

### 4.1 제품 범위 (In Scope)

| 영역 | 포함 내용 |
|------|-----------|
| 탐색 | 홈 히어로·인사이트·데일리 에디트, 대화형 AI 검색(서버 또는 클라이언트 폴백) |
| 필터 | 브랜드(검색+다중 선택), 색상+멀티, 가격대, 카테고리; 오버레이 패널; 필터 칩 UI |
| 상품 | 목 데이터 기반 카드·상세 패널·위시 추가 |
| AI 옷장 | 마이 아이템/위시 탭, 선택 기반 채팅 |
| 계정 | 데모 로그인, 프로필 폼(로컬 상태), 위시리스트 전용 뷰(그리드) |
| 결제 | 체크아웃 UI·모의 완료 |

### 4.2 비목표 (Out of Scope) — 현 단계

| 항목 | 설명 |
|------|------|
| PG·실결제 | 실제 카드/간편결제 연동 없음 |
| 실시간 재고·주문 API | 목 데이터·클라이언트 로직 수준 |
| 관리자·CMS | 미포함 |
| 다국어 | 한국어 UI 기준 |

### 4.3 성공 지표 (KPI) — 제안

| 지표 | 정의 | 비고 |
|------|------|------|
| 상세 진입률 | 대화 세션당 상품 상세 패널 클릭 수 | 로그 수집 시 |
| 필터 사용 | 필터 적용 후 재검색 또는 상세 전환 비율 | |

## 6. 정보 구조 (IA) · 주요 화면

- **홈:** 히어로, 검색(대화 화면 이동), 인사이트 카드, 데일리 에디트 그리드.
- **대화:** 좌측 채팅, 우측 추천 3컬럼 + 필터 바, 관련 상품 스크롤.
- **AI 옷장:** 마이/위시, 그리드 선택, AI 채팅.
- **상품 상세:** 우측 패널, 옵션·리뷰·AI 등.
- **프로필:** 기본 정보·취향·예산 등.
- **체크아웃:** 모의 완료.
- **위시리스트:** 저장된 아이템 그리드(이미지·가격·상세 진입).

## 7. 기능 요구사항 (요약)

### 7.1 AI 검색·대화

- `/api/ai-search` POST; 실패 시 상황·키워드 폴백으로 `MOCK_PRODUCTS` 매칭.
- 관련 상품 갤러리, 퀵 칩 재검색.

### 7.2 필터

- 브랜드·색상(멀티 포함)·가격·카테고리; 패널은 트리거 버튼 기준 정렬(브랜드는 버튼 좌측 기준).
- 외부 클릭 시 닫힘; 필터 칩은 연회색 필 스타일.

### 7.3 상품·데이터

- `MOCK_PRODUCTS` 및 위시 항목 스키마(브랜드, 가격, 색상 옵션, `category` 등).

### 7.4 상품 상세·부가

- 위시, 가격 추적 모달, 체크아웃 CTA.

### 7.5 AI 옷장

- 마이/위시 선택 후 채팅; API 또는 폴백.

### 7.6 계정·프로필

- 로그인 모달, 프로필 필드 로컬 상태.

### 7.7 비기능

- 반응형, 이미지 `referrerPolicy`, API 실패 시 폴백 메시지.
